# Durham Fire Department Staffing Analysis (2021–2026)

## Overview
This project analyzes staffing trends for the Durham Fire Department from **2021 through 2026**. The goal is to understand staffing levels, workforce distribution, and operational trends over time to support better planning and decision-making.

The analysis was performed using **Python** for data processing, cleaning, and analysis, and **Power BI** for data visualization.

---

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib/Seaborn as applicable)
- Power BI (Dashboards & Interactive Visualizations)
- Excel/CSV datasets (source data format)
- Data cleaning and transformation pipelines in Python

---

## Time Range
- Data covers: **2021 – 2026**
- Latest updates (2026 data) are maintained and available within the **City of Durham official datasets and reporting systems**.

---

## Project Objectives
- Analyze staffing levels across multiple years
- Identify trends in personnel allocation and availability
- Highlight potential staffing gaps or surpluses
- Support operational planning and resource allocation

---

## Workflow

### 1. Data Collection
- Staffing data collected from City of Durham records
- Structured into yearly datasets (2021–2026)

### 2. Data Cleaning & Processing (Python)
- Handled missing values and inconsistencies
- Standardized date and staffing categories
- Aggregated data for trend analysis

### 3. Analysis (Python)
- Year-over-year staffing comparisons
- Trend identification
- Summary statistics and key metrics

### 4. Visualization (Power BI)
- Interactive dashboards
- Staffing trend charts
- Department-level breakdowns
- Yearly comparison views

---

## Key Outputs
- Staffing trend analysis (2021–2026)
- Visual dashboards for decision-makers
- Insights into staffing fluctuations and patterns

---

## Notes
- This project is intended for internal analysis and planning support.
- 2026 data reflects the most recent updates available from City of Durham records at the time of analysis.
